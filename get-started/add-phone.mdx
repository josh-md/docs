## Add a New Phone Number

1. Navigate to Settings by clicking your profile icon in the top right corner. Than select `Settings` from the dropdown
2. Select `Phone Management` from the side navigation
3. Select `Add Phone Number` 

![CleanShot 2025-08-25 at 15.04.05.png](/images/CleanShot2025-08-25at15.04.05.png)

After clicking `Add Phone Number` you'll see the following dialog. You can select any option.
